Prevent DHCP Nameservers

isp dhcp nameserver

According to how-to-stop-isp-dns-server-hijacking, ISP will redirect you to an ad website. I will do something to stop this.

  1. Edit /etc/dhcp/dhclient.conf. I have a problem when use “supersede domain-name-servers 127.16.8.1, 8.8.8.8;”, ubuntu only use 127.16.8.1 rather than both nameservers.

    interface “eth0” { supersede domain-name-servers 8.8.8.8; }

  2. Edit /etc/resolvconf/resolv.conf.d/head, if you ifdown eth0; ifup eth0, ubuntu will use the content in this file first. Then use the nameserver in dhclient.conf. I added my own nameserver to this file.

    nameserver 127.16.8.1

Install Ruby

use apt-get to install ruby [ubuntu 14.04]

ubuntu 14.04 provide ruby1.9 and ruby2.0

  1. Use ruby1.9

    If want to ruby1.9 just default package will be fine.

    apt-get install ruby ruby-dev

  2. Use ruby2.0

    Because ubuntu 14.04 default use 1.9, it will be tricky if you want to use 2.0 .

    apt-get install ruby2.0 ruby2.0-dev

    The problem is which gem ubuntu is using, ubuntu has gem1.9 and gem2.0. Default /usr/bin/gem is linked to /usr/bin/gem1.9, so you need to relink /usr/bin/gem to /usr/bin/gem2.0.

    Then install rake and bundler. And notice don’t use apt-get to install rake and bunlder. (If you did, you need to relink them too.)

    rm /usr/bin/gem

    ln -s /usr/bin/gem2.0 /usr/bin/gem

    gem install rake bundler

use rbenv to install ruby [ubuntu 14.04]

There are some patchs need to apply during early period of using ubuntu 14.04. Because a newer version of readline library is used in ubuntu 14.04, some readline functions in ruby source code is deprecated, thus failed to compile. This problem is solved now.

You can choose rbenv or rvm. Generally rbenv is simpler. Rbenv use ruby-build to compile and install ruby versions

  1. install rbenv

    git clone https://github.com/sstephenson/rbenv.git ~/.rbenv

    echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile

    echo 'eval "$(rbenv init -)"' >> ~/.bash_profile

  2. install ruby-build as a rbenv plugin

    git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build
